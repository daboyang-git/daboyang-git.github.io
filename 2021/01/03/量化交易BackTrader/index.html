量化交易BackTrader

一、安装
从github下载源码：
https://github.com/mementum/backtrader
cmd进入目录，输入：python setup install

二、快速开始手册
http://www.backtrader.cn/index.php/backtrader/8-quickstart

三、内置指标
参考手册：http://www.backtrader.cn/index.php/backtrader/11-backtrader

四、相关概念
1.Data Feeds：数据反馈。数据默认传递个策略函数，以array的形式。以self.datas为strategy类默认的变量。
缩写：self.data为self.datas[0]
self.dataX为self.datas[X]
函数调用时如果省略self.datas变量，默认调用self.datas[0]
指标Indicators和操作Operations结果也是DataFeeds，能够继续作为DataFeeds调用。

2.参数：
参数以tuple和dict形式进行传递。参数使用时以self.params的形式使用，简化为self.p

3.Lines：
Lines是一系列数的集合。
示例：
class MyStrategy(bt.Strategy):
params = dict(period=20)
def init(self):
self.movav = btind.SimpleMovingAverage(self.data, period=self.p.period)
def next(self):
if self.movav.lines.sma[0] > self.data.lines.close[0]:
print(‘Simple Moving Average is greater than the closing price’)
lines可以简化：
Xxx.lines为xxx.l
Xxx.lines.name 为xxx.lines_name
line名称可以直接访问，例如：self.data.close和self.movav.sma
Self.line为self.lines[0]
当创建Indicator时，必须声明lines。
Lines len方法，显示多少bars被处理过；buflen显示datafeed共计多少个bars。

4.分片
示例：myslice = self.my_sma[-1:0] # last value and current value

五、平台操作
strategy和indicator都是line迭代器。这个迭代器和python自带的迭代器类似，有如下方法：
1.next（）
2.prenext()
3.nextstart()
4.once()
5.preonce()
6.oncestart()
